# Podcast App

## Introduction

Podcast App is a modern, feature-rich web application built with React and TypeScript. It allows users to discover, listen to, and manage their favorite podcasts with ease. This app provides a seamless and intuitive user experience for podcast enthusiasts, with features like show browsing, episode playback, favorites management, and listening progress tracking.

## Features

- Browse a wide range of podcast shows
- Sort shows by title (A-Z or Z-A) and update date (newest or oldest)
- Filter shows by genre
- View detailed information about each show, including seasons and episodes
- Listen to podcast episodes with a persistent audio player
- Mark episodes as favorites and manage your favorite list
- Track listening progress for each episode
- Responsive design for optimal viewing on various devices

## Technologies Used

- React 18
- TypeScript
- React Router v6
- Tailwind CSS
- Vite (as the build tool)
